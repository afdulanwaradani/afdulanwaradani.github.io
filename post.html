Cara Mendapatkan Domain github.io Gratis

Cara mudah dan cepat mendapat domain gratis dari github.io

Posted by Adani on September 24, 2021

Kali ini adani akan membagikan tutorial dan cara mendapatkan domain github.io. Github adalah media aplikasi yang sangat populer atau biasa kita kenal dengan VCS (Version Control System), Yakni git. Git adalah server repository yang berguna untuk menyimpan dan mempublisakikan kode-kode program dari berbagai versi. Alasan menggunakan Github? karena github merupakan media gratis bagi para developer sebagai media repository full gratis.

Cara Mendapatkan Domain Github.io

Cara mendapatkan cukup mudah, Yakni kalian harus memiliki Akun Github terlebih dahulu.

kemudian membuat repository dengan nama yang sama dengan username Akun Github kamu. jika bingung bisa melihat page ini ya.

setelah semua membuat pagenya, langkah selanjutnya yakni kalian bisa membuat sebuah static html atau mendownload static html. jika anda tidak ingin ribet bisa menggunakan static html di repositori afdulanawaradani git template static html

jika sudah maka lakukan git pull dan git push ke repository main kamu sebelumnya.

Note: proses ini biasanya memakan sedikit waktu dan pikiran.

Kesimpulan

dengan menggunakan domain ini kita tidak perlu lagi untuk memikirkan cara memperpanjang atau membayar perpanjangan domain dengan catatan yakni domain github.io ini merupakan sub domain dari github. jangan hawatir.

kita tetap bisa menggunakan dan juga menambahkan sebuah iklan seperti adsense, MGID, Criteo, Twitter Ads, Amazon Ads dan lainnya. Karena meski sub domain dari github namun bisa berdiri sendiri sesuai keinginan kita.
